帮助选择CSS嵌套的语法

嵌套是Sass等工具里的一个非常受欢迎的功能。它可以节省Web开发者的时间，而不必一遍又一遍地编写相同的选择器上。嵌套还可以让代码更整洁、更容易理解。

无嵌套的CSS
.class1 {
  color: green;
}
.class1 .class2 {
  border: 5px solid black;
}
Sass中的嵌套
.class1 {
  color: green;
  .class2 {
    border: 5px solid black;
  }
}

每个人都希望CSS嵌套可以使用与Sass相同的简单语法。然而因为浏览器解析引擎的工作方式，这并不可能实现。如果他们看到像`element:pseudo`这样的序列，他们将解析整个样式规则，就好像它是一个`property: value`声明一样。
